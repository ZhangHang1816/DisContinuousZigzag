#### 使用说明

1. 需要输入轮廓的边界和参考角度，此边界为闭合边界。
2. 如果为不带孔的边界，则轮廓应该为逆时针
3. 如果填充带孔的轮廓，需要保证轮廓的顺序为外逆内顺：
   建议：外轮廓做内偏置，内轮廓做外偏置
